Remittance/V1 (API) (1.0.0)

# API type Details
1 Remittance/V1/Create The API is to enable you to process your remittance request.
2 Remittance/V1/Find The API is to enable you to get the latest result of your remittance request.
3 Remittance/V1/Search The API is to enable you to get the latest result of your remittance request.
4 Remittance/V1/Notification We will notify the latest status to your system based on the notification URL in Remittance/V1/Create I/F.
Also we will repeat that you until you respond HTTP status 200 and Http response as { "resultType":"suceess" }.

Remittance/V1/Notification

We will notify the latest status to your system based on the notification URL in Remittance/V1/Create I/F.

Also we will repeat that you until you respond HTTP status 200 and Http response as { "resultType":"suceess" }.

We notify you up to 20 times.

The interval expands by 5 minutes.

If we notify 3 times, the 3rd one is notified 15(5*3) minutes later since the 2nd one.

The last 20th notification is notified 1050 minutes(17.5 hours) later since the 1st one.

Retry Interval(minutes) Time from the first try(minutes)
0 - -
1 5 5
2 10 15
3 15 30
4 20 50
... ... ...
19 95 950
20 100 1050
