Request and Response Specification​

Response Specification​

If we get response from agency, we will return it without modification. If we do not get it or there is any error in this API, we will return JSON string with error information.

Response Headers Specification​

All HTTP response headers from agency will be added as response headers of this API. HTTP status code will also be added as a response header, "AGENCY_HTTP_STATUS_CODE".
