Capture API​

Currently, we don't provide the Capture API for PayPal. When we use it, AuthorizeAndCapture API or HTML API are preferred.

CancelOrRefund API​

Payment MethodDetails
PayPalCancel
  • Cancellation can be processed before the authorized transaction expires in 30 days.
Refund
  • Refund/PartialRefund can be processed before the refund expires in 180 days from the timing of being SETTLED status as default. It can be expanded up to 365 days.
If a refund request happens but is not acceptable for PayPal for some reason, the shopper must make a claim. And you will need to issue a manual refund at the PayPal site, such as using the Send Money feature.

The following parameters on API response are optional, and you need to ask PGW to enable them with reasonable reasons if needed on your side.

1.hasMemberId () : true or false if the shopper has MemberID or not.
2.memberId () Contact us if you need it. We will need to check if you have the privilege of getting it. The value is encrypted.
3.hasCvvToken: true or false, if cvv is set with the associated payment transaction
