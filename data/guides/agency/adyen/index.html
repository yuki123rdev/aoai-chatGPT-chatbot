Capture API​

Payment MethodDetails
Credit CardCapture is acceptable up to 28 days(4 weeks) after authorization.

CancelOrRefund API​

Payment MethodDescriptions
Credit CardCancel/Refund
  • There is no limit on the PGW system side to call the CancelOrRefund API, but the shopper's card account still needs to exist for the card issuer to accept the refund request.

Modify API​

Payment MethodDescriptions
Credit cardIf you would like to Modify API with Adyen, your Adyen merchant account needs to turn on the auto-capture setting on the Adyen Customer Area.
It is not specified by Adyen. As long as the shopper's card account is available for Modify, it would work.

Find API​

Reference​

Value on ReferenceDescription
"reference":{
    " hasCvvToken ":false
}
The following parameters on API response are optional, and you need to ask PGW to enable them with reasonable reasons if needed on your side.

1.hasMemberId ( * ) : true or false if the shopper has MemberID or not.
2.memberId ( * ) Contact us if you need it. We will need to check if you have the privilege of getting it. The value is encrypted.
3.hasCvvToken: true or false, if cvv is set with the associated payment transaction
