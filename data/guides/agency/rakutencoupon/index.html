Supported coupon types​

There are two types of Rakuten Coupons, as below.

#TypeDescription
1Discount AmountThe discount amount is fixed. You can set a specific discount amount, such as 500 yen.
2Discount RateLike a 5% discount off or 30% discount off coupon.
It might be complicated to calculate caused of the rounding of fractions.

Notes

  • Rakuten Coupon supports multiple coupons and this allows you to use several coupons at once. However, PGW does not support this feature.
  • The cost of the Rakuten Coupon itself is charged to the Rakuten service using the Rakuten Coupon, not to merchants.

CancelOrRefund API​

Payment MethodMemo
Rakuten CouponCancellation of coupon is available until the order closing (fixed) date.
e.g.) if your service's order closing (fixed) date is the last days of next month(翌月の月末).
#Transaction dateOrder closing (fixed) dateThe time limit for cancellation
FromTo
12017-07-012017-07-302017-08-312017-08-31 23:59:59.999
22017-08-012017-08-312017-09-302017-09-30 23:59:59.999
After the order closing (fixed) date, the used Rakuten Coupon can't be cancelled anymore. However, the used main payment method, like a credit card payment and Rakuten Points redemption, is still cancelable.
  • The CancelOrRefund API returns success as a response, and the transaction itself is cancelled/refunded, but the used Rakuten Coupon is not cancelled.
  • If your transaction's status is kept "authorized", you will not be charged. However, it cannot be cancelled eventually, depending on the closing date of your setting.

Rakuten Coupon​

Please issue coupon on suzaku by yourself.

  1. Set up rakuten stg proxy

  2. Access STG suzaku and login : https://collabo.rakuten.co.jp/adfs/input

  3. Apply for the coupon manager tool

  4. After the admin approved your application, you will see the tool on the main menu.

  5. Select the service which are using our payment system, and click create new coupon

  6. Currently (13/03/2019) we only support service coupon, so please select couponType = "Service Coupon" and Discount Applicable target = "Service"
    After input the mandatory field, click "SAVE", and then click "COMPLETE"
