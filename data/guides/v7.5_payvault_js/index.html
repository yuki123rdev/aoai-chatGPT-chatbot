Installation​

Include the payvault.js version in your checkout page HTML

For those clients who upgrade to 7.5

from 7.5, the color validation only allows lowercase letter color names, so you may get an error in the setup method. Please change your CSS color style to the lowercase letter

Staging​

Version 7.5
<script type="text/javascript" src="https://stg-static-content.payment.global.rakuten.com/pv/payvault/V7/7.5/payvault.js"><script/>

Production​

Version 7.5
<script type="text/javascript" src="https://static-content.payment.global.rakuten.com/pv/payvault/V7/7.5/payvault.js"><script/>

CVV length validation improvement​

Although in a very rare use case, some users enter their CVV first, then their card number.

In that case, the number of digits in CVV may not be checked correctly.

This version will check correctly the length even in a such case.
