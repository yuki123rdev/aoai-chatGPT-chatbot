Installation​

Include the payvault.js version in your checkout page HTML

Staging​

Version 7.4
<script type="text/javascript" src="https://stg-static-content.payment.global.rakuten.com/pv/payvault/V7/7.4/payvault.js"><script/>

Production​

Version 7.4
<script type="text/javascript" src="https://static-content.payment.global.rakuten.com/pv/payvault/V7/7.4/payvault.js"><script/>

Display Card Brand Logo​

This feature behaviour is same as payvault.card.getBrandCode method, If you have set brandCodes which doesn't allow your customer to use, then the card icon will show default one.
For knowing more information about card band rule, please check here. Card Number Validation Rule
note: the size of card icon depends on the font size in different style statuses.
If you're interested in card icons, please check with ReX.

Changelog​

2022/06/01

  • Add card icon and cvv icon
    • support Visa, MasterCard, Diners Club, Discover, JCB, American Express

2022/03/31

  • Change API to Add V4
    • This new API won't require any extra configuration to set it up, but if you request an extra configuration about the country code you should create a special ticket to make the configurations, click here for more information Information Add V4
  • Logging System
    • This new feature will help us to keep error log from the client side so will be more easy find out issues with PayVault Js
