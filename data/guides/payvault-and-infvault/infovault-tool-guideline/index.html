Introduction​

Purpose of this document​

This document presents what InfoVault Tool is and what you can do with InfoVault Tool. If you are new to InfoVault Tool, you must first get new account in How to create new account.

About InfoVault Tool​

InfoVault Tool is used to check the sensitive file you added previously with file token or requestId.

Caveats
  1. IE8/9 or previous versions should not be used to access this tool.
  2. The user will be blocked automatically after 7 times failed attempts. If you need to unblock the user, please contact pro-payvault-all@mail.rakuten.com with your account information.

URLs​

The following URLs are globally accessible from any device connected to the Internet. No proxy settings are required.

EnvironmentURLDefaultAvailableRemarks
Staginghttps://infovault-tool.stg.rpsp.jp/Login pageYes
Productionhttps://infovault-tool.pro.rpsp.jp/Login pageYes but only from allowed IP address

Key Features​

Search Function​

The function is designed to search what you registered previously in InfoVault. Currently, you are able to search only file itself and related information.

To search file with requestId in request URL​

  1. Open InfoVault Tool at https://infovault-tool.stg.rpsp.jp?requestid="dfdsfasdfdfasdf".

  2. Type Username and Password. infovault-login

  3. Click Login

To search file without any parameter​

  1. Open InfoVault Tool

  2. Type Username and Password. infovault-login

  3. Click Login

To search file without any parameter​

  1. Open InfoVault Tool
  2. Type Username and Password.

infovault-login

  1. Click Login and set the following values on Search File page.
    a. File Token: The file token you got when you add file previously.
    b. Request ID:The Request ID you issued when you add file previously. In case if you set both File Token and Request ID, File Token will be used for search.
    c. Show File:This checkbox is used to control if user wants to download the file or not. If the user has the relevant permissions then the checkbox will be selectable; by default the checkbox is checked.

infovault-screen-home

  1. Click Search then the file and related information will be shown on right side of the page infovault-file-token

To delete file​

  1. Open InfoVault Tool

  2. Type Username and Password. infovault-login

  3. Click Login and set the following values on Search File page.
    a. File Token: The file token you got when you add file previously.
    b. Request ID:The Request ID you issued when you add file previously. In case if you set both File Token and Request ID, File Token will be used for search.
    c. Show File:This checkbox is used to control if user wants to download the file or not. If the user has the relevant permissions then the checkbox will be selectable; by default the checkbox is checked.

infovault-screen-home

  1. Click Search then the file and related information will be shown on right side of the page

infovault-file-token

  1. If the user has the permission a delete button will be visible on the lower right hand side of the image information panel.

Please be careful from this step

こちらの手順より、丁寧に実施してください。

infovault-file

  1. A final confirmation message will appear if the user pressed the delete button on the previous step. a. Delete - Once confirmed this will permanently delete the file.
Before pressing Delete 押す前に

If you pressed the "Delete" Button on the popup screen, the file will be permanently deleted.

ポップアップの"Delete"ボタンを押すと、ファイルは完全に削除されます。

It is irreversible.

このアクションは不可逆です。

b. Close - Closing the confirmation will cancel the delete. infovault-delete-diagram

Change Password Function​

The function is designed to enable user to change login password.

To change password​
  1. Open InfoVault Tool. a. Type Username and Password. infovault-login

b. Click Login and choose Settings → Change Password.

  1. Type Current Password, New Password and Repeat Password. a. Current Password: The current password you are using now. b. New Password and Repeat Password: The new password you want to use. infovault-change-password

  2. Click Change Password
